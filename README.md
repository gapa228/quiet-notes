# Тихий ежедневник

Устанавливаемый личный ежедневник для компьютера и телефона. Работает без интернета, хранит дела, дни рождения, подписки, покупки и расходы, а после подключения облака синхронизирует их через личный аккаунт.

## Что уже работает

- главный экран «Сегодня», выбор даты, создание, редактирование, поиск, закрепление и удаление заметок;
- повторы каждый день, неделю, месяц или год и отметка выполнения;
- отдельные типы записей: дела, дни рождения, подписки и продукты;
- главная только с актуальными напоминаниями и отдельный каталог купленных продуктов;
- настраиваемый интервал покупки или оплаты, например каждые 2 недели;
- стоимость и история расходов с отчётом за текущий месяц;
- автоматическое локальное сохранение;
- офлайн-режим и установка на домашний экран;
- регистрация и вход по email;
- синхронизация между устройствами по принципу «последнее изменение побеждает»;
- изоляция данных: каждый пользователь видит только свои записи.

## Подключение синхронизации

1. Создайте бесплатный проект на [Supabase](https://supabase.com/).
2. Откройте **SQL Editor**, вставьте содержимое `supabase.sql` и нажмите **Run**.
3. В **Project Settings → API** скопируйте **Project URL** и **anon public key**.
4. Вставьте их в `config.js`:

```js
window.NOTES_CONFIG = {
  supabaseUrl: "https://ВАШ-ПРОЕКТ.supabase.co",
  supabaseAnonKey: "ВАШ_ANON_KEY"
};
```

5. Опубликуйте папку на любом HTTPS-хостинге (например, Cloudflare Pages, Netlify или GitHub Pages). Откройте полученную ссылку на обоих устройствах и войдите с одним email.

Важно: `anon public key` предназначен для размещения в браузере. Безопасность заметок обеспечивают политики RLS из `supabase.sql`. Никогда не помещайте в `config.js` ключ `service_role`.

## Локальный просмотр

PWA-функции требуют адрес `http://localhost` или HTTPS. Запустите любой статический веб-сервер в этой папке и откройте показанный адрес в браузере. Простое открытие `index.html` двойным щелчком позволит проверить интерфейс, но установка и офлайн-кэш работать не будут.
